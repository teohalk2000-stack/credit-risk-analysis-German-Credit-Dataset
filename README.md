# Credit Risk Data Analysis Project

## 📊 Project Overview
This project analyzes the German Credit dataset to identify factors associated with loan default risk.

## 🛠️ Tools Used
- SQL
- Google Sheets
- Statistical Analysis (t-test, chi-square)

## 🔍 Key Steps
- Data cleaning and feature engineering
- SQL-based analysis
- Statistical hypothesis testing

## 📈 Key Insights
- Higher default risk among younger applicants with short employment history
- Statistically significant difference in loan amounts between good and bad credit
- No strong relationship between installment rate and default risk
